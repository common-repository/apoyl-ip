== Description ==

实现发布文章记录IP地址，用户发布文章显示IP归属地，评论显示IP归属地,更加方便了解用户来自哪里.

## 插件功能

* 支持发布文章记录IP地址，
* 支持发布文章显示IP归属地（比如来自 上海）  +
* 支持评论显示IP归属地（比如来自 北京）
* 归属地以系统查询结果为准
* 采用固定的存放IP归属地到数据库中，好处比如即使IP归属地发生变化对之前用户也不会影响，减少查询消耗
